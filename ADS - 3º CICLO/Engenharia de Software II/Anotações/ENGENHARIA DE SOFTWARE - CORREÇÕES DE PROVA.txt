** CORREÇÃO - SISTEMA ACADÊMICO

  -- AGENDAMENTO DE CONSULTAS

    + PROBLEMAS
     ++ 
   
    - ATORES

     + PACIENTE
      ++ CADASTRAR PACIENTE
      ++ GERENCIAR AGENDAMENTO ( REAGENDAR CONSULTAS )
      ++ VISUALIZAR HISTÓRICO DE CONSULTAS 
      ++ CONFIRMAR PRESENÇA
      ++ EFETUAR PAGAMENTO

     + ATENDENTE
      ++ CADASTRAR PACIENTE ( CADASTRAR | CONSULTAR | ATUALIZAR DADOS ) 
      ++ GESTÃO DE AGENDAMENTO ( CONSULTAR AGENDA | AGENDAR CONSULTA PARA PACIENTE )
      ++ VISUALIZAR HISTÓRICO
      ++ GERAR RELATÓRIOS

     + MÉDICOS 
      ++ VISUALIZAR AGENDA
      ++ REGISTRAR INFORMAÇÕES
      ++ VISUALIZAR HISTÓRICOS

     + SISTEMA
      ++ VALIDAR LOGIN E AUTENTICAÇÃO
      ++ NOTIFICAR PACIENTE 
      ++ INTEGRAR COM SISTEMA DE PAGAMENTO 
      ++ REGISTRAR HISTÓRICO DE AÇÕES 

    + CASO DE USO - PROBLEMA GERAL 
     ++ EFETUAR O LOGIN 

------------------------------------------

** EXEMPLO EM AULA - DIAGRAMA CASO DE USO 

  -- ATORES
   + PACIENTE
   + ATENDENTE
   + MÉDICO
   + SISTEMA

 
  -- CASO DE USO

   + EFETUAR LOGIN
    -> ATORES: PACIENTE | ATENDENTE | MÉDICO | SISTEMA
    -> include -> VALIDAR DADOS ( sistema )

  -- CASO DE USO - ATORES
  
    + ATENDENTE
     ++ CADASTRAR PACIENTE
     ++ GERENCIAR ATENDIMENTO
       -> extend | include -> NOTIFICAR ( sistema ) 
     ++ GERAR RELATÓRIOS

    + PACIENTE
     ++ VISUALIZAR HISTÓRICO 
     ++ CONFIRMAR PRESENÇA
     ++ EFETUAR PAGAMENTO 
        -> include -> INFORMAR PAGAMENTO

    + MÉDICO
     ++ VISUALIZAR HISTÓRICO
     ++ VISUALIZAR AGENDA
     ++ REGISTRAR INFORMAÇÃO

    + SISTEMA
     ++ NOTIFICAR
      -> extend ->
     ++ LOG DE SISTEMA
     ++ VALIDAR LOGIN
      -> include -> EFETUAR LOGIN ( paciente ) 
     ++ INFORMAR PAGAMENTO 
      -> include -> EFETUAR PAGAMENTO ( paciente )
