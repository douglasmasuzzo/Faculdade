  + PIX 

 -- [ RN2 ] -> REGRAS PARA OPÇÕES DEFINIDAS

  + ATOR ESCOLHE A OPÇÃO

  + CARTÃO -> " CASO DE USO XX : OPÇÃO DE PAGAMENTO POR CARTÃO DE CRÉDITO | DÉBITO É EXECUTADO "  

  + CARTÃO DE DÉBITO -> " CASO DE USO XX : OPÇÃO DE PAGAMENTO POR CARTÃO DE DÉBITO É EXECUTADO "

  + PIX -> " CASO DE USO XX : OPÇÃO DE PAGAMENTO POR PIX ( À VISTA ) É EXECUTADO "  
 
  + PIX -> " CASO DE USO XX : OPÇÃO DE PAGAMENTO POR PIX ( POR PARCELA ) É EXECUTADO "  

 -- [ R3 ] -> VERIFICAR RETORNO DE STATUS

   + SE O PAGAMENTO FOR APROVADO, O SISTEMA APRESENTA MENSAGEM " PAGAMENTO APROVADO "

   + SE O PAGAMENTO FOR NEGADO, O SISTEMA APRESENTA A MENSAGEM " PAGAMENTO REPROVADO ".
