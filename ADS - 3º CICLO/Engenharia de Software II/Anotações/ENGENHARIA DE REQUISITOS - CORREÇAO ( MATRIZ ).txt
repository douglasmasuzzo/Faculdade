** ESPECIFICAÇÃO DE REQUISITOS DO SISTEMA

** CORREÇÃO

  RF001 -> CADASTRO DE USUÁRIO
   + PERMITIR A CRIAÇÃO | EDIÇÃO | REMOÇÃO DE CONTA 

  RF002 -> CONTROLE DE ACESSO
   + DEFINIÇÃO DE AUTENTICAÇÃO

  RF003 -> EDIÇÃO DE PERFIL
   + DESCRIÇÃO DEFINIDA PELO CRUD 

  RF004 -> CONSULTA DE CATÁLOGO
   + CRIAÇÃO DO CATÁLOGO DE UM PRODUTO
   + PERMITINDO A CONSULTA DE FILTROS
   + ALTERAÇÃO DE INFORMAÇÕES

  RF005 -> COMPRA ONLINE
   + GERENCIAR O MÉTODO DE PAGAMENTO COM API'S DIFERENTES
   + CONFIGURAR O MÉTODO DE COMPRA 
  
  RF006 -> SUPORTE AO CLIENTE
   
  RF007 -> HISTÓRICO DE PEDIDOS
   + REFINAR A DESCRIÇÃO 

  RF008 -> AVALIAÇÃO DE PRODUTOS

  RF009 -> NOTIFICAÇÕES DE LANÇAMENTOS
   + REFINAR A DESCRIÇÃO
   + INCREMENTAR A POLÍTICA LGPD
   + PERMITIR A EXIBIÇÃO DE NOTIFICAÇÕES CONFORME A AUTORIZAÇÃO DO USUÁRIO 

  RF010 -> CONTROLE DE ESTOQUE
   + INTELIGAR O MÓDULO DE VENDAS AO MÓDULO DE CONTROLE DE ESTOQUE
   
  RF011 -> GERENCIAMENTO DE PEDIDOS
   + REFINAR A DESCRIÇÃO 
   + ( CRITÉRIO DEFINIDO DA PROFESSORA PARA INCLUIR NA ESPECIFICAÇÃO DO CASO DE USO ( FUNCIONALIDADE )  ) 

  RF012 -> GERENCIAMENTO DE CATÁLOGO
   + REFINAR A DESCRIÇÃO
   + PERMITINDO QUE O USUÁRIO ADMINISTRADOR GERENCIE O CATÁLOGO DE PRODUTOS DA EMPRESA

  RF013 -> CÁLCULO DE FRETE
   + REDEFINIR O REQUISITO ( GERENCIAMENTO DE ENTREGA DO PRODUTO )
   + REFINAR A DESCRIÇÃO
   + " O SISTEMA DEVE PERMITIR QUE O USUÁRIO GERENCIAR O FLUXO FINAL DE ENTREGA " 

  RF014 -> LISTA DE DESEJOS

  RF015 -> RELATÓRIO DE VENDAS
   + REDEFINIR O RF ( ESPECIFICAMENTE DIRECIONADO A SAÍDA )
   
  RF016 -> GERENCIAMENTO DE PROMOÇÕES
   
  RF017 -> NOTIFICAÇÃO DE PROMOÇÕES
   + REDEFINIR O REQUISITO
   + REDEFINIR A DESCRIÇÃO

  RF018 -> CRIAÇÃO DE CATEGORIAS
   + REDEFINIR O REQUISITO

  RF019 -> RASTREAMENTO DE PEDIDOS
   + REFINAR A DESCRIÇÃO

  RF020 -> RECUPERAÇÃO DE SENHA
   + REFINAR A DESCRIÇÃO
