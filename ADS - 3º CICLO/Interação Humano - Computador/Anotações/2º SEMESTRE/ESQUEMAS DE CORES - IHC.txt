IHC - ESQUEMAS DE CORES

----------------------------------------

** ESQUEMA DE CORES

  -- TODOS OS SITES DEVEM TER UM ESQUEMA DE CORES
    + CORES PRIMÁRIAS ( ÁREAS PRINCIPAIS )

  -- O USO REPEDIDO DESSAS CORES AFETARÁ A MENTE E O HUMOR DO USUÁRIO, GERALMENTE DE FORMA INCONSCIENTE

  -- EMBORA HAJA MUITAS MANEIRAS DIFERENTES DE MISTURAR CORES, VAMOS INTRODUZIR 3 ABORDAGENS COMUNS

** ESQUEMA DE CORES - TRIÁDICO

  -- É O MAIS BÁSICO E EQUILIBRADO
  
  -- USANDO VIBRAÇÃO E COMPLEMENTAÇÃO, A ESTRUTURA TRIÁDICA É O ESQUEMA MAIS SEGURO E CONFIÁVEL

  -- SELECIONE QUAISQUER ( TRÊS ) CORES LOCALIZADAS A 120° UMA DA OUTRA
    + UMA COR PARA O SEGUNDO PLANO
    + DUAS CORES PARA O CONTEÚDO E NAVEGAÇÃO

** ESQUEMA DE CORES - COMPOSTO

  -- É UM POUCO MAIS COMPLICADO DE SE REALIZAR, MAS PODE SER RECOMPENSADOR SE FOR BEM EXECUTADO
   
   -- O CONCEITO USA QUATRO CORES
     + DOIS PARES DE MESMA VIBRAÇÃO
     + DOIS PARES DE CORES COMPLEMENTARES

** ESQUEMA DE CORES - ANÁLOGO

   -- DESTACA A VIBRAÇÃO
     + EX: UM ESQUEMA ANÁLOGO ( VERMELHO | LARANJA | AMARELO ) APARECERÁ MAIS ENERGÉTICO E VIVO

   -- FACIL DE REALIZAR, O TRUQUE ESTÁ EM DECIDIR QUAL VIBRAÇÃO DE CORES USAR, SENDO APLICADO EXAGERADAMENTE
