Este é o corpo do primeiro tópico. Ele é exibido por padrão, até que o plugin de colapso adicione as classes apropriadas que usamos para estilizar cada elemento. Essas classes controlam a aparência geral, bem como o mostrar e esconder via transições de CSS. Você pode modificar qualquer um desses itens com CSS personalizado ou sobrescrever nossas variáveis padrão. Vale lembrar que praticamente qualquer HTML pode ser inserido dentro do .accordion-body, embora a transição limite o transbordamento.
